\## Info
[30702 국기 색칠하기](https://www.acmicpc.net/problem/30702)

## 💡 풀이 방법 요약
1. 색만 다르고 모양 같은 국기
2. N-M 모든 구간의 색(A~Z)이 동일하다면 같은 국기
3. 주어진 다른 2개의 국기를 같은 국기로 만들기 위해 색을 칠할 거임
   3-1. 같은 구역 -> 같은 색이면서 인접한 곳
   3-2. 구역을 원하는 색으로 변경 가능
4. 모든 list에 대해 bfsB가 가지고 있으면 YES
