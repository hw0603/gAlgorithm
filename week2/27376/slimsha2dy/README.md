## 💡 풀이 방법 요약

1. k만큼 입력을 받은 후 위치로 정렬
2. 결과로 출력할 변수 res, 현재 위치 tmp 변수 선언
3. 현재 위치에서 다음 신호등까지 거리(x - tmp)만큼 res에 더함
4. 만약 신호등이 동작하지 않았다면 res를 s로 변경
5. 신호등에 도착했는데 빨간불인지 확인: ((res - s) / t)가 2로 나뉘어지면 초록불
    * 다음 초록불이 켜지는 시간으로 res를 변경
6. 3번부터 반복
